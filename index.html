HTML tags Reference

Form Cheatsheet

Tag name Description
<form> form The purpose of a <form> is to allow users to input information and send it.
<input> input To add fields for users to input information we use the <input> element and set the type attribute to a field of our choosing:
<select> select populated with <option> elements and renders a dropdown list selection.
<datalist> populated with <option> elements and works with an <input> to search through choices.
<text area> text area a text input field that has a customizable area.
